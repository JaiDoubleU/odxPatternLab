View Attachments

Field tickets can have one or more attachments associated with each field ticket. Attachments can be viewed from the Field Ticket details page.

Users can see the number of attachments included on a field ticket when viewing a list of field tickets in a stack.

 

Steps:

1.Click on the field ticket to view the field ticket details.

2.Click on Attachments.

Note:The attachments list displayed on a field ticket includes the name, type, and size for each attachment.
