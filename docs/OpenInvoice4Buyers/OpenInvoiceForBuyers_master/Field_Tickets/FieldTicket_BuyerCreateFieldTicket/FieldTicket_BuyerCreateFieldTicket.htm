Buyer Initiated Field Ticket

This feature allows buyers to submit a Field Ticket to themselves on behalf of their registered suppliers. The buyer leverages data from an existing system and creates a field ticket in OpenInvoice using this information. The buyer then approves the Field Ticket using the approve a Field Ticket via API feature. A registered supplier is now able to quickly create an invoice from this approved field ticket.

If you are interested in more information on this feature, please reach out to your account manager. Your account manager can enable the functionality and guide you through the implementation options.

Note:The buyer can only use this feature if their account manager has enabled the preference.
