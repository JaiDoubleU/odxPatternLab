Bulk Actions on Field Tickets

This section describes the bulk actions buyers can apply to field tickets.

Note:Users must have the Field Ticket Bulk Approve permissions in order to see the Approve button.

How to Update Multiple Tickets Using Bulk Edit

The bulk edit feature allows you to edit the names users (Requisitioner, Ticket Owner, Superintendent, and Office Approver), AFE, Cost Center, GL Coding (Major, Minor, Subcode), and any custom fields you have defined.

Important!The Bulk Edit feature is available within the Ticket Application only.

Steps:

  1. Click on Ticket in the navigation bar and select Ticket Application.

    2.  

  2. In the Quick Search field, type a ticket number and click the Search icon to activate the search.

    3. Important!The Field Ticket Quick Search no longer includes an implied wildcard. If you want to search for all tickets that begin with the numbers/letters entered you must include the wildcard (*) character.

  3. Click List to view the field tickets in the list view.

  4. Select the checkboxes next to the tickets you want to modify.

    5.  

  5. Click Bulk Edit. A Bulk Edit Tickets window displays.

    6.  

  6. Do one of the following:

How to Bulk Approve Field Tickets

 The Bulk Approve feature allows you to select multiple field tickets and approve them all at the same time.

Permission:The Enable Bulk Edit company preference must be active to bulk approve field tickets.

Important!The Bulk Edit feature is available within the Ticket Application only.

Steps:

  1. Click on Ticket in the navigation bar and select Ticket Application.

    2.  

  2. Select a Ticket Stack.

  3. Select the checkbox next to the field tickets you want to approve.

    4.  

  4. Click Approve. The Approve Tickets window displays.

  5. (Optional) Enter a comment.

  6. Click Yes, approve tickets. All of the selected field tickets are now approved and are now displayed under the Approved tab.
