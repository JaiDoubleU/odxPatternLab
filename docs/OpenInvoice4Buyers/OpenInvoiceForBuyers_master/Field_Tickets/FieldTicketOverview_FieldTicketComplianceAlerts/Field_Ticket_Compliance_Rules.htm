Field Ticket Compliance Rules

Field Ticket Compliance Rules (workflow rules) allow you to reject Field Tickets on the basis of Supplier Type, Supplier Number, Purchase Order, and Price Book conditions. The compliance rules are configured using an XLSX file.

How to Export Field Ticket Compliance Rules

Steps:

  1. Select Corporate > Workflow Rules to display the workflow configuration rules page.

  2. Click Configure Field Ticket Compliance Rules.

    3.  

  3. Click Export to download the current compliance rules.

    4. Note:If this is the first time you are downloading the XLSX file you will receive a template with the header row containing the fields used to create the rules.

  4. Navigate to the folder on your machine where you want to download the XLSX file.

  5. Click Save.

How to Modify the Field Ticket Compliance Rules

This procedure describes how to download and modify the field ticket compliance rules spreadsheet.

Steps:

  1. Open the FieldTicketComplianceRules.xlsx file in Microsoft Excel on your machine.

  2. Update the rules as needed.

    3.  

  3. Select File > Save to save the updates.

  4. Select File > Close to close the XLSX file.

How to Import the Field Ticket Compliance Rules

This procedure describes how to upload your field ticket compliance rules spreadsheet.

Steps:

  1. Select Corporate > Workflow Rules to display the workflow configuration rules page.

  2. Click Configure Field Ticket Compliance Rules.

    3.  

  3. Click Import.

  4. Select File in the Import Method field.

  5. Click Choose File.

  6. Navigate to the folder on your machine where the XLSX file resides.

  7. Select the FieldTicketComplianceRules.xlsx file.

  8. Click Open.

  9. Click Upload.
