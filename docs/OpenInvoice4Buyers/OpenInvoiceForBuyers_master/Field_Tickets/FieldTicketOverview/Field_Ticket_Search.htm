Field Ticket Search

Field tickets can be found by clicking Search in the OpenTicket menu or entering a ticket number in the Global Search field in the upper right corner of OpenTicket.

How to Search for a Field Ticket

This procedure allows users to search for field tickets.

Steps:

  1. Click on the Menu button and select Search.

  2. Enter one or more search criteria to locate tickets. Entering more than one search criteria will narrow the search results.

    3. §Ticket Number: enter all or part of a ticket number. Use an * if you are entering a partial ticket number.

    §Ticket Status: select Approved, Disputed, Submitted, Cancelled, or leave blank to search for tickets in any Ticket Status

    §Supplier Name: enter a supplier name to view all tickets for the selected supplier. Use an * if you are entering a partial supplier name.

    §Invoiced Status: select Not Invoiced, Partially Invoiced, Fully Invoiced, or leave blank to search for tickets in any Invoiced Status.

    Note:If a field ticket is associated with an invoice the buyer will see in the ticket application “Invoiced: 12345” at the top of the screen, where “12345” represents the invoice number that is associated with that field ticket.

    §(Optional) <Custom Data Fields>: The custom data fields are displayed.

    §Date Criteria: select the type of date you want to search for; such as: Service Date, First Submit Date, or Approved Date.

    §Service Date From: enter the earliest service date. Format is MM/DD/YYYY.

    §Service Date To: enter the latest service date. Format is MM/DD/YYYY.

    §Ticket Approver: the name of the person who approved the field ticket.  

    §Requisitioner: enter the name of the person who can approve the field ticket.

    §Superintendent: enter the name of the person who can be contacted for additional information required to correctly code / approve an invoice referencing this ticket.

    §Office Approver: enter the name of the person responsible for approving invoices referencing a ticket.

    §Ticket Type: select the appropriate ticket type from the drop-down list.

    §Location: type in a specific location to find tickets associated with that location.

    §AFE: type in a specific AFE to find tickets associated with that AFE.

    §Cost Center: type in a specific Cost Center to find tickets associated with that Cost Center.

    §PO: type in a specific PO number.

  3. Click Search.

Search results will appear below the search fields. You can sort search results by any column by clicking on the column header. Click the + in the top right of the search results to customize the columns displayed. Click on the Ticket No. to view a ticket.

Note:If a field ticket is associated with an invoice the buyer will see “Invoiced: 12345” at the top of the ticket, where 12345 represents the invoice number that is associated with that field ticket.

How to Search for a Field Ticket Using Search Field

You can also search for tickets in the top right corner of any OpenTicket page.

Steps:

  1. In the Search field, type a ticket number and click the Search icon to activate the search.

Important!If you want to search for all tickets that begin with the numbers/letters entered you must include the wildcard (*) character.

 

The search results can be viewed in card or list format and can be sorted by specific columns in ascending or descending order.

From the list view you can select the columns you want to view, sort your tickets, change back to card view, or select multiple tickets you want to modify using the new bulk edit feature.

Note:In the List view you can control how many results you see on the page. By default, you will see 25 field tickets on a page. You can change the number of results on a page by updating the Rows per page field at the bottom of the page. The options are: 10, 25, 50, 100, 250, or 500. To move through the pages, use the page controls in the bottom right corner of the page. When using the Bulk Edit feature, selecting the checkbox at the top of the page will ONLY select the fields tickets displayed on this page, not all field tickets in the search results.

Note:The columns drop-down list includes any custom fields you have defined.
