Field Tickets

Once products and/or services are delivered, the supplier can create and submit a ticket to the buyer. This section provides an overview of how users can view and manage tickets in OpenInvoice.

Field Ticket Application

To access tickets in OpenInvoice click Ticket on the navigation bar and select Ticket Application.

 

Users will be taken into the Field Ticket Application.

 

The field ticket application is accessible from desktop and mobile devices. There are four sections of the OpenTicket main page.

  1. Menu: In the top left corner users will find a menu of options:

    2.  

  2. Field Ticket Search: Search for a ticket by a ticket number. Type the ticket number in the Search field and click the Search icon. Users can also search for tickets using the Field Ticket Search function in the desktop version of OpenInvoice.

    3. Note:Ticket search will return the first 500 qualifying tickets. Refine your search by adding more specific search parameters.

    Important!The Field Ticket Quick Search no longer includes an implied wildcard. If you want to search for all tickets that begin with the numbers/letters entered you must include the wildcard (*) character.

  3. Stacks: View previously created stacks of tickets. Each stack displays the number of submitted tickets in the stack. Click on the number to access tickets that have been submitted, approved or disputed in the stack. Tickets may appear in more than one stack. Additional information on creating and managing ticket stacks can be found in the Create Stacks section.

    4. Note:Ticket stacks will return the first 500 qualifying tickets.

  4. Create New Stacks: Create new stacks of tickets by AFE, Cost Center, Location, Requisitioner or Supplier, or a combination of these fields. Additional information on creating and managing ticket stacks can be found in the Create Stacks section.
