Create Stacks

Buyers can organize tickets by creating stacks similar to how tickets are organized on a buyer’s desk. Each user can have unique stacks of tickets in his/her OpenInvoice account. Field tickets can be stacked by:

·AFE

·Cost Center

·Custom Data Field

·Location

·Requisitioner

·Supplier Name

·Ticket Type

Each stack is then grouped by status.

Note:When a new ticket user navigates to the ticket application the system automatically creates a ticket stack showing tickets where the requisitioner is the user. The stack is named “<user> Tickets” and ticket notifications are turned on.

How to Create a Stack

This procedure describes how to create a stack in the Field Ticket application.

Steps:

  1. Click Ticket > Ticket Application. The Field Ticket application displays.

  2. Do one of the following:

    3. §If there are no stacks created, click on the Create Stack button in the middle of the Ticket Stacks page.

    §If stacks have already been created, click on the + sign in the top right corner of the page.

    Note:The Ticket Search page displays.

  3. Enter the search criteria.

  4. Click Search. The search results display.

    5.  

    Note:The option to save a stack is now available even when there are no tickets that meet the criteria. This will provide you with the ability to create a stack in advance of tickets being submitted to that project and/or requisitioner.

  5. Click Save As, then click New Stack.

    6.  

  6. Enter the Stack Name. This is your descriptive name for the stack.

    7. Note:The Stack Names should be short.

  7. (Optional) Select the Notify Me checkbox if you want a daily email notification that provides a report of the age of the tickets in the stack.

  8. Click Save to create the stack. A “Success” message displays briefly at the top of your screen.

 

The blue number represents the number of tickets in each stack that require processing or review. If there are zero tickets to process or review you can click the blue number to view all tickets in a stack. Tickets may appear in more than one stack.

 

The page will display all tickets in the stack (in either List or Card format). The tickets are displayed by ticket status. Click on a ticket to view the ticket details and attachments. The field tickets can be viewed in card or list format.

How to Edit a Stack

This procedure describes how to edit the settings for an existing stack in your Field Ticket application.

Steps:

  1. Click Ticket > Ticket Application. The Field Ticket application displays.

  2. Click the Options icon in the upper right corner of the stack box.

  3. Select Settings.

  4. Update the search fields as necessary.

  5. Click Search. Verify the search results reflect the changes you want.

  6. Click Save As, then click Existing Stack to update the current stack.

How to Delete a Stack

This procedure describes how to delete an existing stack in your Field Ticket application.

Steps:

  1. Click Ticket > Ticket Application. The Field Ticket application displays.

  2. Click the Options icon in the upper right corner of the stack box.

  3. Select Delete. A confirmation window displays.

    4. Note:Deleting a stack will not delete the Field Tickets associated with the stack.

  4. Click Delete Stack.
