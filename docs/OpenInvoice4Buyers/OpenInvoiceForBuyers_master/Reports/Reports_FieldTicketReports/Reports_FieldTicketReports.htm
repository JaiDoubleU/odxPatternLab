Field Ticket Reports

Field ticket reports, both summary and detail, can be generated from the OpenTicket Search page or within the Ticket Application. The Detailed Report allows users to extract field ticket details so the data can be manipulated by another system, without requiring users to manually re-key the information. This report can be used for a manual import into a reporting system, an excel file for data analysis, or importing into a cost reporting system such as Wellview or Openwells.

Field Ticket Summary Report

The Summary Report results return one row per field ticket with header information (in XLS file format). This report provides summary information and not all the data in the field ticket.

Field Ticket Detail Report

The Detail Report results return one row per ticket line (in XLS file format). This report provides all the data for the field ticket.
