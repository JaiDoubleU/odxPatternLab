Modifying a Department’s Notification User

As users change job functions you may need to update the notification user for a department.

How to Modify the Notification User for a Department

Use this procedure to modify the notification user for a department.

Permission:Membership Admin

Steps:

  1. Select Membership > Departments & Roles.

  2. Click the department name.

  3. Click the Settings tab.

  4. Click the department name.

    5.       

     

  5. Modify the user in the Notification field.

    6. Note:To search for a specific user, enter all or part of the user's name and click the search icon.

    To select from a list of all users in your company, enter an asterisk (*), then click the search icon.

  6. Click Submit.
