Granting Financial Authority

Granting financial authority determines which invoices a user is able to financially approve based on your company’s configuration, the total amount of the invoice, the currency of the invoice, and the general ledger codes applied to the invoice. Financial authority is determined by the authorization levels assigned to the user.

How to Modify the Authorization Levels Assigned to Users

To modify authorization levels for a single user:

Permission:Membership Manage Auth to assign authorization levels to other users. 

Note:You must have the Self Manage attribute enabled on your profile to assign authorization levels to yourself.

Steps:

  1. Select Membership > Users.

    2. Note:By default all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Click either the last name or first name of the user.

  3. Click the Authorizations tab.

    4.       

          

  4. Do one of the following:

§To assign an authorization level, select an approval type, currency, and authorization level from the drop-downs, then click Set Authorization Level.

§To remove an authorization level, click the delete icon then OK to confirm.

To modify authorization levels for multiple users:

Permission:Membership Manage Auth to assign authorization levels to other users. 

Note:You must have the Self Manage attribute enabled on your profile to assign authorization levels to yourself.

Steps:

  1. Select Membership > Authorization Levels.

  2. Click the edit icon in the Users column.

    3.       

          

     

  3. To grant or remove the authorization level to users, add or remove users from the Users Configured box by selecting a user and clicking Add < or > Remove.

    4. Note:To select multiple users, hold the Ctrl key and select the users. Alternatively, you can hold the Shift key to select a range of users.

  4. Click Submit.
