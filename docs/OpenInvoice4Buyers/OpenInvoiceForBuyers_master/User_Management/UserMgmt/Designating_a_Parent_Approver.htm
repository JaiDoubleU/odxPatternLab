Designating a Parent Approver

If the user cannot perform a final financial approval on an invoice, he or she must forward it to another user, such as a parent approver, for further financial approval. If a parent approver is not designated, the user may forward the invoice to a site and department and/or user that can perform a final financial approval.

How to Set a Parent Approver

Permission:Membership Manage Auth to assign the parent approver for other users. 

Note:You must have the Self Manage attribute enabled on your profile to modify your parent approver.

Steps:

  1. Select Membership > Users.

    2. Note:By default, all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Click either the last name or first name of the user.

  3. Click the Authorizations tab.

    4.       

     

          

  4. In the Invoices requiring further approval get forwarded to drop-down, select a user.

  5. Click Set Parent Approver.
