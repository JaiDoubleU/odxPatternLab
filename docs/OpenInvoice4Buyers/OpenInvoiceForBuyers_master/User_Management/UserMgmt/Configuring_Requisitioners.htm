Configuring Requisitioners

Depending on your company’s configuration, a requisitioner may or may not be required to be listed on a document. To configure whether a requisitioner must be listed on your suppliers’ invoices, Select Requisitioner.

To be listed as a requisitioner on a document, the requisitioner does not need access to the OpenInvoice Platform; however, a user profile must be created for them. To be listed as a requisitioner on a document, the user must:

·be in any user status except Inactive

·be assigned the OpenInvoice license

·have the Invoice View permission

·be assigned an authorization level

To add a user, refer to Configuring User Access.

To assign a license, refer to Modify the Licenses Assigned to a User.

To assign permissions, refer to Roles.

To assign an authorization level, refer to Granting Financial Authority.
