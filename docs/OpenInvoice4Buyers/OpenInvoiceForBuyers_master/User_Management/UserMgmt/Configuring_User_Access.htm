Configuring User Access

To access the OpenInvoice Platform and modify their profile, a user must have:

·a valid and active user profile

·completed the registration process to create a password and security questions

Note:Users must be assigned the Self-Manage attribute to manage their own permissions and roles.

In order to work with documents and/or modify your company’s configuration, the user must have at least one role with permissions and at least one license.

How to Add a User

Use this procedure to add a new user.

Permission:Membership Modify

Steps:

  1. Select Membership > Users.

    2. Note:By default, all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Click Add User.

    3.  

  3. Click the Self-Manage checkbox.

  4. Enter the Username, First Name, Last Name, and Email address. All other fields are optional.

    5. Note:The username must be unique within the entire OpenInvoice Platform (not just your company). Usernames must be between 8 and 32 characters. Email address format is acceptable.

  5. Decide if the registration email should be sent now or later. To send the email later, deselect the Send Registration Email checkbox.

  6. Do one of the following:

    7. §Click Submit - to save this user

    §Click Add Another - to add another user

  7. Click the Organization Roles tab.

  8. Click the + next to the site name to expand the site information.

    9.  

  9. Select the checkbox in the left column for the role you want to assign to the user.

  10. Click Submit.

The new user will receive an email with the registration link.

How to Modify the Roles Assigned to a User

Use this procedure to modify the roles assigned to a specific user.

Permission:Membership Modify to assign roles that do not have Admin permissions.

Membership Manage Auth to assign roles that have the Membership Manage Auth permissions.

Membership Admin to assign any other role.

Membership Modify and the Self-Manage attribute to assign roles that do not have Admin permissions to yourself.

Membership Admin and the Self-Manage attribute to assign any role to yourself (excluding the Membership Manage Auth permissions).

Note:Users can be assigned to a role when adding or editing a role.

Steps:

  1. Select Membership > Users.

    2. Note:By default, all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Click either the Last Name or First Name of the user.

  3. Click the Organization Roles tab.

    4.  

  4. Expand one or more sites.

  5. Select and deselect the checkboxes to assign or remove roles from the user.

    6. Note:The user must be added to the department’s Member role.

  6. Click Submit.

Note:Users affected by changes to role assignments will retain their current permissions until they sign out. Any altered access to the platform will take effect the next time they sign in. Users can be forcibly signed out if necessary.

 

How to Modify the Licenses Assigned to a User

Use these procedure to modify the licenses assigned to a single user or multiple users.

To modify the licenses assigned to a single user:

Permission:Membership Modify

Note:A user must be assigned at least one license to sign in to the OpenInvoice Platform.

Steps:

  1. Select Membership > Users.

    2. Note:By default, all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Click either the Last Name or First Name of the user.

  3. Click the Licenses tab.

    4.  

  4. Select or deselect the checkboxes to assign or remove the licenses from the user.

  5. Click Submit.

Note:Users affected by changes to license assignments will retain their current access level until they sign out. Any altered access to the platform will take effect the next time they sign in. Users can be forcibly signed out if necessary

To modify the licenses assigned to multiple users:

  1. Select Membership > Product Licenses.

  2. Click the license product name.

    3.  

  3. Do one of the following:

    4. §To assign the license to users, select one or more Non-Licensed users and click Add >.

    §To remove the license from users, select one or more Licensed users and click < Remove.

  4. Click Submit.

How to Initiate User Registration and/or Account Reset

To access OpenInvoice, users must first complete the registration process. They are sent an email with a link to a process that walks them through creating a password, agreeing to the terms of use, confirming their profile information, and defining their security questions. This process is also used to grant access to users who are suspended or require an account reset.

To send the registration email to a single user:

Use this procedure to send the registration email to a single user.

Permission:Membership Modify

Prerequisite:You have provided the user with their username.

Steps:

  1. Select Membership > Users.

    2. Note:By default, all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Click either the Last Name or First Name of the user.

  3. Click the Summary tab.

    4.  

  4. Click either the Send Registration Email or Reset Account button.

    5. Note:The appropriate button appears based on the user’s account status.

  5. Click OK to confirm.

Note:When resetting a user account, the account is inaccessible until the user completes the process.

You cannot send the reset account email to yourself.

When registering, users have 14 days to complete the process before the email link expires and the registration email must be sent again.

Important!When registering, if a user does not enter their username correctly on the first attempt, the email link becomes invalid and the registration email must be sent again.

To send the registration email to multiple users:

Use this procedure to send the registration email to multiple users.

Permission:Membership Modify

Prerequisite:You have provided the user with their username.

Steps:

  1. Select Membership > Users.

    2. Note:By default, all users are listed. Apply a filter to narrow the list. If only part of the criteria is known, an asterisk (*) must be used to denote the unknown part of the criteria.

  2. Select the checkbox next to the users’ names.

    3.  

  3. Click Send Registration Email.

  4. Click Yes to confirm.

Note:When resetting a user account, the account is inaccessible until the user completes the process.

You cannot send the reset account email to yourself.

When registering, users have 14 days to complete the process before the email link expires and the registration email must be sent again.

Important!When registering, if a user does not enter their username correctly on the first attempt, the email link becomes invalid and the registration email must be sent again.
