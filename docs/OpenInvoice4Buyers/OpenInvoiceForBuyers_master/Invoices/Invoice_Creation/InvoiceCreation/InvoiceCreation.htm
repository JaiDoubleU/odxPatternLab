Duplicate Invoice Number Checking

OpenInvoice checks for duplicate invoice numbers prior to invoice submission. The following criteria must be met before the invoice can be submitted:

·The invoice number must be unique between the buyer company and supplier company (a supplier cannot submit an invoice number that has been previously submitted to the same buyer).

·The invoice number must be unique among supplier sites that share the same supplier number (a supplier cannot submit an invoice number that has been previously submitted by another site or supplier company who has the same supplier number assigned by the buyer).

If the buyer company preference Enable Invoice Number Masking is enabled, both originally-entered invoice numbers and system-generated masked invoice number are subjected to duplicate invoice number checking. Invoices with a duplicate number cannot be submitted and will fail to export.

Invoice numbers restrict the special characters to prevent accidental duplication of invoices. The allowed special characters are:

! " # $ % & ' ( ) * + , - .  / : ; < = > ? [ \ ] ^ | _ ` { | } ~

Creating an Invoice from an Image File

There are two ways to create Image OBO invoices. The accepted file types differ in the two methods.

·Batch: multiple image files are uploaded then indexed one at a time. Only image files (.jpg, .jpeg, .gif, .png and .tiff) and PDF files can be uploaded when batch processing. This method may be used to add new suppliers as part of the invoice indexing process. If the user is unable to complete the supplier setup, the invoice is put on-hold until a user with the Corporate Partnership permission is able to complete the process.

·Single: An image file is uploaded and indexed on the page where you select the supplier you are working with. PDF, Word, spreadsheet, jpg, .jpeg, .gif, .png, and .tiff image files are supported.

Invoices created on behalf of a supplier can be itemized at any point in the invoice workflow. For instructions on how to itemize an invoice, refer to Itemizing an Image Invoice.

How to Upload and Index Image Invoices

Permission:Invoice Modify (IM)

Steps:

1.Select Invoice > Dashboard (under the Create Invoice column of menu).

2.Enter the supplier name, number, postal/zip code, or site name to display a list of matching suppliers.

3.Select an entry from the list that appears as you type.

Note:Only the first 10 matching entries are displayed in the list.

4.Click Select.

5.Select Invoice > Create Invoice (under the Create Invoice column of menu).

Note:The Create Invoice submenu item is only available when you are acting on behalf of the supplier. Above the main navigation bar, the name of the supplier will be displayed next to your name. You are no longer working on behalf of the supplier after you navigate outside of the Create Invoice module.

6.Select a customer site, and department.

Note:A field will automatically populate if it only has one option.

      

 

7.Enter the Invoice Number and invoice Description.

8.Select a Currency.

Note:You can begin typing the name of the currency to narrow the list of selections as you type. This field will remember your last selection within your current session.

9.Enter the Service Date.

10.Enter the Amount.

11.(Optional) Enter or modify the Invoice Date and Routing Code.

Note:The Invoice Date will default to the current date.

The Amount is the total value of the invoice after all discounts and taxes have been applied. Do not include any Early Pay Discount amounts.

12.To upload an image file, click Upload Image File, then Choose File. Navigate to the image file on your computer, click Open, then click Upload.

13.(Optional) Modify the Image File Description.

14.Click Next.

15.Do one or more of the following:

§Click Edit Header to modify the invoice header information.

§Click Contact to add a contact to this invoice.

§Click Add Header Data.

Note:This option is only available if the buyer has configured additional data fields for the invoice header.

§Click Itemize to itemize the invoice.

§Click Comment.

Note:This option is only available if the invoice has been saved at least once.

§Click Attach File.

16.Do one or more of the following:

§Click Save to save the invoice. You can submit the invoice at a later time.

§Click Submit to submit the invoice into workflow immediately.

Note:Upon submission, OpenInvoice will check to ensure the invoice number is unique. If a duplicate invoice number is found, the invoice cannot be submitted and an error message will appear.

How to Upload Multiple Image Files

Permission:Invoice Modify (IM)

Steps:

1.Select Invoice > Image Invoice Upload.

2.Click Select Files, navigate and select the image files on your computer, then click Open.

Note:To select multiple items, hold the Ctrl key and select the items. Alternatively, you can hold the Shift key to select a range of items. If the files are in multiple folders, upload the files from one folder at a time.

3.Click Upload Files.

 

How to Index Multiple Image Files

Permission:Invoice Modify (IM)

Note:This process is used to index and submit invoices in the Working Queue and Exception Queue.

Steps:

1.Do one of the following:

§Immediately after uploading image invoice files on the Image Invoice Upload page, click Image Invoice Processing.

§Select Invoice > Image Invoice Processing.

Note:The first invoice in the working queue will automatically open in a new window.

2.(Optional) Apply a filter to find the appropriate image file.

Note:Image files that have not been indexed and submitted to workflow are automatically deleted after 99 days. The Days To Deletion field shows the number of days remaining.

3.Click an image file from the Working Queue tab.

 

4.Do one or more of the following:

§Click the icon in the Send to Exception Queue column if the invoice cannot be processed
at this time. You may enter a comment that will appear next to the file name in the Exception Queue tab.

§Click the Delete icon, enter a Comment, and click Submit.

§Index and submit the invoice using the steps below.

5.(Optional) Enter a PO number.

Note:This will populate data on the form with the corresponding information from the purchase order.

By default, if valid purchase orders exist in OpenInvoice for the supplier, the referenced purchase order must be one of the purchase orders in platform.

6.Select a supplier company and site:

§Enter a supplier name, number, postal zip/code, or site name.

§Select an entry from the list that appears as you type.

Note:Only the first 10 matching entries are displayed in the list. Click More to view the complete list of matching suppliers.

7.If necessary, create a new supplier:

§Click Add Supplier.

§Enter the supplier name and any other information that is known.

§Click Submit.

8.Select a Customer Site and Customer Department.

Note:A field will automatically populate if it only has one option.

9.Enter the Invoice Number, Invoice Date, Description, and Total Amount incl. Tax Amount.

The Amount is the total value of the invoice after all discounts and taxes have been applied. Do not include any Early Pay Discount amounts.

10.Select a Currency.

11.(Optional) Enter Tax Type/Tax Amount.

 

Note:To enter more than one tax type/tax amount, click the Add button.

12.(Optional) Modify the Service Date and/or Routing Code.

13.(Optional) Enter one or more Ticket No.

14.(Optional) Click Itemize.

15.Click Submit.

16.Repeat steps 2 through 16 to index another uploaded file.

Itemizing an Image Invoice

Itemization breaks the total invoice amount into separate lines with the product/service description, quantity, unit of measure, rate, and any applicable discounts. Itemization adds detail lines to the invoice allowing for the same level of reconciliation that can be performed on a traditional invoice. The only difference is that the itemized invoice does not reference a supplier’s catalog in the OpenInvoice Platform.

How to Itemize an Image Invoice

This section provides the procedures for itemizing image invoices.

To itemize an invoice that is not yet submitted:

Permission:Invoice Modify (IM) for Image OBO or Image Post invoices

Invoice Modify for Image Scan invoices not itemized by the supplier

Note:OpenInvoice will delete your itemization details if the supplier changes the invoice amount on a disputed Image Scan invoice that you have itemized. The Invoice Journal will display a message stating line item details have been automatically removed.

1.Do one of the following:

§Select Invoice > Dashboard (under Create Invoice column) and click one of the dashboard counters.

§Perform a search by invoice number using the Global Search field in the main navigation bar.

§Perform a search with multiple criteria by going to Invoice > Invoice Search.

§Itemize an invoice on the Image Invoice Processing page before submitting the invoice.

2.Click Itemize.

 

3.In the Itemization Wizard window, enter the invoice information.

Note:If the invoice references a purchase order in the platform, the Itemization Wizard will use the purchase order as a template.

4.Enter the product/service information in the lines.

§Click in a field to make the entire row editable. Use the Tab key to move from left to right or Shift+Tab to move from right to left.

§Use the Enter key to move to the first column of the next row. The Enter key also adds a new row if the user is currently on the last row.

§Use the arrow keys to move up and down in the same column.

Note:The down arrow key also moves control to the buttons when on the last row.

5.In each line, enter the Description, Quantity, and Rate. All other fields are optional.

Note:Enter 1 in the Qty field for tax lines.

6.(Optional) The itemization wizard assumes the line item will have taxes applied. If a line item does not require taxes, select the Exclude Tax checkbox. This option applies to Image OBO invoices.  

7.Click Submit.

Additional Information

·The Submit button in the Itemization Wizard will not be available until the Total Amount field equals the exact amount of the invoice.

·If the Itemization Wizard is accessed from the Image Invoice Processing page, the invoice can be submitted after itemization. If the Itemization Wizard is accessed when viewing the image invoice, the invoice must be saved before any further actions can be taken. 

·If the itemization is changed, all prior coding is deleted (except for any short pay lines). Whenever the Itemization Wizard is opened and Submit is clicked, the coding must be redone even if no changes have been made. To close the Itemization Wizard without deleting prior coding, click Cancel.

·Tax lines must be added for any applicable taxes that were on the invoice.

·If the purchase order line is a detail line, then:

§The invoice line Description is populated using the buyer’s description from the purchase order line.

§The invoice line Unit of Measure and Rate are populated with the corresponding information on the purchase order line.

§The Quantity is defaulted to the remaining fulfillment quantity for the purchase order line.

·If the purchase order line is a limit order line, then:

§The invoice line Description is populated using the buyer’s description from the purchase order.

§The invoice line Unit of Measure, Rate, and quantity information are not populated

Itemization Wizard Details

·Number of Lines: number of lines on the invoice including tax lines and invoice level discounts.

·Enter line discounts as: select whether any applicable line item discounts should be applied by Amount or Percentage. All lines will have the same type of discount.

·Error: an icon will appear if there is an error in an entry. Click the icon to see an explanation of the error.

·Discount: may be a percentage or amount depending on the option selected in the Enter line discounts as field.

Note:Do not enter Early Pay discounts here.

Image Search

Search for image files that have been uploaded, deleted, and invoiced. You can search by file name, date range, and/or document status.

Results will display the file name, status, status date, invoice number (if the file has been invoiced), and comments (if any). The Action column allows you to view or download your file. If you have the Invoice View permission for an invoice you can view it by clicking on the invoice number. Configure your search result columns by clicking the plus icon next to the Action column.

How to Search for Image Files

Permission:Invoice Modify (IM)

Steps:

1.Select Invoice > Search Images (under the Create Invoice column).

2.Enter your search criteria.

Note:If only part of the file name is known, an asterisk (*) must be used to denote the unknown part of the name.

3.Click Search.

Exporting Image Files

To analyze and report on images uploaded for processing in OpenInvoice, users can export image files in the Working and Exception queues on the Image Invoice Processing page. Images are exported into a TSV file.

How to Export Image Files

Permission:Invoice Modify (IM)

Steps:

1.Select Invoice > Image Invoice Processing (under the Create Invoice column).

2.Next to the Working Queue and Exception Queue tabs, click the Export Lists button.

3.Select the queues you want to export.

 

4.Click Export.

5.Click Save to save the TSV file.
