Transaction Upload

Buyers can import previously approved and/or paid financial transactions for payment and reporting purposes. This section provides instructions on how to import transactions.

·Upload Methods

Uploaded Transactions Overview

Imported transactions are assigned the Transaction Upload submission type. They can be associated with any enabled supplier (registered or unregistered) and can be imported into any active buyer site and department including those that are not flagged as a Remit To department.

Transactions can be uploaded with either an Approved or Paid status. These invoices cannot be unapproved. OpenInvoice will attempt to generate and schedule a payment for the transaction if:

·The transaction was uploaded with an Approved status.

·The buyer and supplier are configured for payments.

·The buyer is configured to generate payments on invoice approval.

·The transaction currency is USD.

The transaction upload will fail after the first occurrence of the following errors:

·The specified supplier is not found or not enabled.

·The transaction cannot be assigned a single supplier site.

·The specified buyer site and department are not active.

·A duplicate transaction is found.

Note:If an error occurs, all transactions in the file will fail to upload. The upload process stops after the first error is found and will not provide feedback on any additional errors that may exist in the file.

Upload Methods

Transaction files can be manually uploaded into the platform or uploaded electronically through integration.

How to Manually Upload a Transaction

Permission:Transaction Upload

Steps:

1.Select Invoice > Transaction Upload.

2.Click Export Template to download and save a copy of the template.

3.In the file, modify the information as necessary. For more information about the column headings, refer to Transaction Upload Template  .

4.Do one of the following:

§To upload a file saved on your desktop, select File as the import method, click Choose File, navigate to the file on your computer, and click Open.

 

§To copy the contents of a spreadsheet, select Copy/Paste as the import method, copy the information in your file, and paste it into the box.

5.Click Upload.
