Transfer your back officeâ€™s AFEs, cost centers, locations, major codes, minor codes, subcodes, work orders, and associated business rules. Unlike the AFE Import and Cost Center Import, the Cost Collection Import file defines the relationships between objects.

Cost Collection messages are SOAP wrapped. The maximum size of each message is 3 MB. Larger files may time out and result in a failed import.

The Cost Collection Import is not a scheduled service. Instead, you will post the files as necessary. The file is not saved on the secure server but is, instead, redirected to the OpenInvoice Platform if not posted there directly.

Schema: CostCollection.xsd (v. 1.0)
