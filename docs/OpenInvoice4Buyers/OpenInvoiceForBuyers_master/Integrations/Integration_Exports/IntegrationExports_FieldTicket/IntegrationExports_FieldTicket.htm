Field Ticket Export

Field tickets are transferred to the buyer’s back office. Individual charges cannot be exported unless they are part of the Field Cost Export. If a supplier enters a standalone charge in OpenInvoice, the system will create a ticket for it before it is exported.

Field Ticket Exports are transmitted in MIME format.

This service will only run if the buyer has the OI Ticket Workflow company preference 2 Step Approval Workflow enabled.

·Schedule: can be run at specified time(s) every day.

·Include All Attachments: If set to Yes, any attached backup documentation will be exported as well.

·Attachment/PDF Directory: Location for any attachments.

·Filemask: The location where the file will be sent and name of the created file.

Scheduled Service: Field Ticket Export

Schema: PIDX Field Ticket 1.2
