Purchase Order Response

When a supplier accepts or rejects a purchase order in the OpenInvoice platform, a PIDX Purchase Order Response document is sent to the buyer if the Purchase Order Ack Export service is enabled.

Messages are SOAP wrapped.

·Schedule: can be run at specified time(s) every day or in real time (purchase order acknowledgments are exported within a few minutes of being accepted or rejected in the OpenInvoice interface).

·Filemask: The location where the file will be sent and name of the created file.

Scheduled Service: Purchase Order Ack Export

Schema: PIDX Order Response 1.2
