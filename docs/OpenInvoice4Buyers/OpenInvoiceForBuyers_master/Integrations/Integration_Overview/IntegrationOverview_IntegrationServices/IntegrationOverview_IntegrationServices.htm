Integration Services

The Integration Services page can only be accessed by users with the Corporate Integration permission. There are four portlets on this page that help users to manage their company’s integration activities:

·The Service Monitoring portlet summarizes the status of each service.

·The Integration Services Notifications portlet displays messages about integration activities.

·The Services portlet lists services with active or manual schedules. From this portlet, you can manually run a service, view service configuration, and pause or resume all services.

·The Notification Users (Imports) lists the users that will be notified when a problem occurs on an import. From here, you can add or remove notification users.

      

 

The Service Monitoring Portlet

The Service Monitoring portlet allows users to quickly view the status of integration services. Service names are listed with the number of pending, failed, or completed exports. Click on a number to view its associated list of documents. The columns in this portlet can be configured by clicking the plus icon in the top right corner of the Service Monitoring portlet’s scroll bar.

The Integration Services Notifications Portlet

To view messages about integration activities Corporate Integration users must subscribe to the Integration Services message type by going to Membership > My Profile. Subscribing to the preference will automatically send messages to the portlet as well as your Message Center. You may also choose to receive a copy of the messages by email. Messages in the Integration Services Notifications portlet will be automatically deleted after two weeks.

Note:A user who has unsubscribed from the Integration Services preference may still receive messages in the portlet if they have initiated an export directly from the user interface.

The Services Portlet

The Services portlet allows users to manually run a service, view service configuration, and pause or resume all services. Services requiring manual export are indicated in the Schedule column. Click the action icon to select one of the options available for the service.

The Notification Users (Imports) Portlet

The Notification Users (Imports) portlet allows users to configure service notification users. Only users with the Corporate Integration permission can be selected as notification users.

How to Manually Select Invoices for Export

Permission:Corporate Integration

Steps:

  1. Select Corporate > Integration Services.

  2. In the Services section, click the service’s action icon.

  3. Select the Select Invoices option.

  4. Enter and select criteria for the invoices you wish to export.

    5. Note:Your criteria is saved for future searches.

  5. Click Find Invoices.

  6. Select the checkboxes next to the invoices you want to export.

  7. Click Export Selected.

How to Manually Run a Service

Permission:Corporate Integration

Steps:

  1. Select Corporate > Integration Services.

  2. In the Services section, click the service’s action icon.

  3. Select Run.

How to View Service Settings

Permission:Corporate Integration

Steps:

  1. Select Corporate > Integration Services.

  2. In the Services section, click the service’s action icon.

  3. Select View.

  4. Click Cancel to return to the Integration Services page.

How to Pause and Resume Services

Permission:Corporate Integration

Steps:

  1. Select Corporate > Integration Services.

  2. In the Services section, click Pause Services or Resume Paused Services.

Note:While services are paused, a warning message will appear at the top of the page.
