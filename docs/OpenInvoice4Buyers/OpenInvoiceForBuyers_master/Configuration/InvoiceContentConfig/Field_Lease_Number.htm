Field/Lease Number

A field/lease number is used to track expenditures or time against a property or asset (warehouse, patent, serial number, etc.). The number is created externally from OpenInvoice but is added by going to Corporate > Reference Data.

How to Make the Field/Lease Number a Required Field

These procedures require the Corporate Admin permissions.

Enable the Restrict the Field/Lease by AFE or Cost Center Field Preference

Steps:

  1. Select Corporate > Company Preferences.

  2. Click Restrict Field/Lease by AFE or Cost Center under the Invoice Content section.

  3. Select the Active checkbox.

  4. Click Submit.

Specify the Invoice Workflow Step That Requires the Field/Lease

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Field/Lease.

  5. Select the appropriate workflow.

    6.  

  6. Click Submit.

How to Configure Supplier Access

This procedure configures whether a supplier can view or modify Field/Leases on an invoice.

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Field/Lease.

  5. Select the appropriate Supplier Access option from the drop-down list.

  6. Click Submit.

How to Configure Buyer Access

This procedure configures whether users within your company are able to view or modify Field/Leases on an invoice.

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Field/Lease.

  5. Select the appropriate Buyer Access option from the drop-down list.

  6. Click Submit.
