Cost Center

Cost centers and cost center types are added by going to Corporate > Reference Data. Buyer and supplier access to the Cost Center field is configured by going to Corporate > Workflow Configuration.

Note:When configuring invoice data fields, setting the AFE and Cost Center fields to None will also hide the property information on the invoice.

Relevant Company Preferences

·Require AFE or Cost Center

·PO Coding Override

How to Make the Cost Center a Required Field

These procedures require the Corporate Admin permissions.

Enable the Require AFE and Cost Center Fields Preference

Steps:

  1. Select Corporate > Company Preferences.

  2. Click Require AFE or Cost Center under the Invoice Content section.

  3. Select the Active checkbox.

    4. Note:This preference has no effect on B2B suppliers.

  4. Click Submit.

Configure GL Codes to Require a Cost Center

Use this procedure to configure certain general ledger account codes to require a Cost Center.

Steps:

  1. Select Corporate > COA Setup.

  2. Click Coding Validation.

  3. Click Account Type.

  4. Click Cost Center.

  5. Select the Major and Minor accounts you want to add.

    6.  

  6. Click Add To Required.

  7. Click Submit.

Configure GL Codes to Require Specific Cost Center Types

Steps:

  1. Select Corporate > COA Setup.

  2. Click Coding Validation.

  3. Click Cost Center Type.

  4. Click the Cost Center Type you want to configure.

  5. Select the Major Accounts and Minor Accounts.

  6. Click Add >.

  7. Click Submit.

Configure AFE to Require a PO

Steps:

  1. Select Corporate > AFEs.

  2. Search for an AFE Number.

    3. Note:To search for all AFE Numbers, type * in the AFE Number field and click Apply Filter.

     

  3. Click the AFE Number you want to configure.

  4. Select Required in the PO Required drop-down list.

  5. Click Submit.

Specify the Invoice Workflow Step That Requires a Cost Center

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Detail tab.

  4. Click Cost Center.

  5. Select the appropriate workflow.

    6.  

  6. Click Submit.

Reject Invoices Without a Cost Center

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Configure Invoice Workflow Rules.

    3. Note:B2B invoices that are rejected by the invoice workflow rules will be automatically disputed on your behalf.

  3. Import the Invoice Workflow Rules file following the Invoice Workflow Rules Template.

How to Configure Supplier Access

This procedure configures whether a supplier can view or modify Cost Centers on an invoice.

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Cost Center.

  5. Select the appropriate Supplier Access option from the drop-down list.

  6. Click Submit.

How to Configure Buyer Access

This procedure configures whether users within your company are able to view or modify Cost Centers on an invoice.

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Cost Center.

  5. Select the appropriate Buyer Access option from the drop-down list.

  6. Click Submit.

Validations Based on other Data Elements

·Restrict which cost centers can be specified on an invoice based on the selected buyer site by going to Corporate > Cost Centers.

·Restrict which cost centers can be specified on an invoice based on the selected AFE by going to Corporate > AFEs.

·If a cost center is specified on a purchase order line and the PO Coding Override preference is turned off, that cost center must be on the invoice line referencing the purchase order line. If the PO Coding Override preference is enabled, the cost center defaulted from the purchase order can be modified.
