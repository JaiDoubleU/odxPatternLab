Work Order Number

A work order number identifies a request for products and/or services. The number is created externally from OpenInvoice when your company creates a work order.

How to Make the Work Order a Required Field

These procedures require the Corporate Admin permisssions.

Specify the Invoice Workflow Step That Requires a Work Order

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click WO.

  5. Select the appropriate workflow.

    6.  

  6. Click Submit.

How to Restrict Work Order Values to a Defined List

By default the Work Order field is a free-text field that will accept any value. You can restrict your work orders to a predefined list of values.

Add Work Order Values

Steps:

  1. Select Corporate > Reference Data.

  2. Click WO List Type.

  3. Click Add Record.

    4.  

  4. Enter the Work Order list information.

  5. Click Submit.

Configure Work Orders to a Defined List

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click WO.

    5.  

  5. Select List from the Data Type drop-down list.

How to Configure Supplier Access

This procedure configures whether a supplier can view or modify WOs on an invoice.

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click WO.

  5. Select the appropriate Supplier Access option from the drop-down list.

  6. Click Submit.

How to Configure Buyer Access

This procedure configures whether users within your company are able to view or modify WOs on an invoice.

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click WO.

  5. Select the appropriate Buyer Access option from the drop-down list.

  6. Click Submit.
