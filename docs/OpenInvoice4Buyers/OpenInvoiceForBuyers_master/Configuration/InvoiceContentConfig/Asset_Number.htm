Asset Number

An asset number identifies an asset within your company. It is created externally from OpenInvoice for asset reporting and accounting.

How to Make the Asset Number a Required Field

These procedures require the Corporate Admin permissions.

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Asset Number.

  5. Select the appropriate Required option from the drop-down list.

    6.  

  6. Click Submit.

How to Configure Supplier Access

This procedure configures whether a supplier can view or modify Asset Numbers on an invoice.

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Asset Number.

  5. Select the appropriate Supplier Access option from the drop-down list.

  6. Click Submit.

How to Configure Buyer Access

This procedure configures whether users within your company are able to view or modify Asset Numbers on an invoice.

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Asset Number.

  5. Select the appropriate Buyer Access option from the drop-down list.

  6. Click Submit.
