Location

Cost objects can be associated with one or more locations. If an AFE is associated with a default location, the Location field on invoices referencing the AFE will be automatically populated with the associated location. It is left blank if the AFE is not associated with a location. Disable the Restrict Location by AFE or Cost Center preference if allowing the supplier to edit the field.

How to Make Location a Required Field

These procedures require the Corporate Admin permissions.

Enable the Restrict Location by AFE or Cost Center Field Preference

Steps:

  1. Select Corporate > Company Preferences.

  2. Click Restrict Location by AFE or Cost Center under the Invoice Content section.

  3. Select the Active checkbox.

  4. Click Submit.

Configure the GL Codes to Require the Location

Steps:

  1. Select Corporate > COA Setup.

  2. Click Coding Validation.

  3. Click Account Type.

  4. Click Location.

  5. Select the Major and Minor accounts you want to add.

  6. Click Add To Required.

  7. Click Submit.

Specify the Invoice Workflow Step That Requires the Location

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Location.

  5. Select the appropriate workflow.

    6.  

  6. Click Submit.

How to Configure Supplier Access

This procedure configures whether a supplier can view or modify the Location on an invoice.

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Location.

  5. Select the appropriate Supplier Access option from the drop-down list.

  6. Click Submit.

How to Configure Buyer Access

This procedure configures whether users within your company are able to view or modify the Location on an invoice.

  1. Select Corporate > Workflow Configuration.

  2. Click Invoice Data Fields.

  3. Click the Line Item Coding tab.

  4. Click Location.

  5. Select the appropriate Buyer Access option from the drop-down list.

  6. Click Submit.
