Configuring Unregistered Suppliers

By default, Check is selected as the payment method for the supplier. To pay the supplier electronically using ACH, you must configure the supplier’s banking details.

IMPORTANT!  If the Bank Details or Remit To address for a supplier is modified, then all users with the Invoice Admin and/or Payment Admin permissions will receive an email notification of the change.

How to Configure Bank Details and Payment Method

Permission:Corporate Partnership and Payment Admin

Steps:

  1. Do one of the following:

    2. §Go to Payment > Dashboard to select from a list of suppliers requiring set up.

    -Click either the Not Configured for Payments or Incorrect Configuration counter.

    §Go to Corporate > Suppliers to search for the supplier or select from a list of existing suppliers.

    -In the Tasks portlet, click Search Suppliers.

    -In the Suppliers portlet, click a counter.

  2. If necessary, apply a filter to narrow the list of suppliers.

  3. Click the supplier site.

  4. On the Supplier Configuration page, click Edit in the Banking Details section.

  5. Do one of the following:

§To pay the supplier by check, select Check as the Payment Method.

§To pay the supplier electronically by ACH:

  1. Select ACH as the Payment Method.

  2. Select the ACH Type from the drop-down list.

  3. Enter the Financing ABA Number and Financing DDA Number.

    4. Note:The ABA number must be 9 digits.

  4. Click Submit.

  5. Click Done.

Site Banking Details

·Financing ABA Number: contains 9 characters and is required for ACH payments.

·Financing DDA Number: is required for ACH payments.

·Bank Country Code: payments through OpenInvoice Payment must always have a US country code.

·Account Type: will be Checking or Savings.

·Payment Method:

§Check: OpenInvoice’s money movement partner will print and mail a physical check to the Remit To address.

§ACH: ACH (Automated Clearing House) is the electronic money transfer mechanism used by OpenInvoice’s money movement partner. This method can only be used for US companies transacting in USD. There are two different ACH methods: CCD+, which supports a single payment per transaction and CTX which supports multiple payments per transaction.

How to Configure Remit To Information and Validate for Payment

Permission:Corporate Partnership and Payment Admin

Note:Remit To and Validate for Payment information can also be configured using file upload. Refer to the instructions, “How to Manage Unregistered Supplier Sites via File Upload”, in the “Supplier Management” section of the OpenInvoice User Guide for Buyers.

Steps:

  1. Go to Payment> Dashboard.

  2. Click the Not Configured for Payments or Incorrect Configuration counter.

  3. If necessary, apply a filter to narrow the list.

  4. Click the supplier site.

  5. On the Supplier Configuration page, click Edit in the Remit To Address section.

  6. Enter the Pay-To Name, Address, City, Country, State, and ZIP/Postal Code. All other fields are optional.

    7. Note:Use the Address 1 and Address 2 fields. The Address 3 and Address 4 fields do not apply to payments.

    The zip code must be in xxxxx or xxxxx-xxxx format.

  7. Select the Validate For Payment checkbox.

  8. Click Submit.

  9. Click Done.

Site Contact Information

·Remit To Address: the primary address displayed on all documents.

§Pay-To Name: the name used by OpenInvoice to process your company’s payments. If the supplier is paid by check, this is the name that will appear on the check. If the supplier is paid electronically, this is the name associated with their bank account.

§Enter a different Legal Address: select this checkbox if the legal address and contact person differ from the Remit To Address and Remit To Contact.

§Enter a different Correspondence Address: select this checkbox if the correspondence address and contact person differ from the Remit To Address and Remit To Contact.

·Remit To Contact: the supplier site’s contact information (name, email, phone number, and fax number).
