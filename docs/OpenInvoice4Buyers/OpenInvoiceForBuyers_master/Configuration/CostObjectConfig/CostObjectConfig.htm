Cost Object Configuration

This section provides instructions on how to configure your company’s cost objects. Cost object management may include specifying properties, adding locations, and/or associating with sites.

The following topics are applicable to company administrators who oversee configuration of the cost allocation functionalities in OpenInvoice:

·Managing AFEs

·Managing Cost Centers

·Associating Locations to a Cost Object

·Associating a Cost Object with a Site

Managing AFEs

To view your AFEs, select Corporate > AFEs and perform a search. If criteria is unknown, click Apply Filter to return a complete list of AFEs.

How to Add an AFE

Permission:Corporate Modify

Steps:

1.Select Corporate > AFEs.

2.Click Add AFE.

 

3.Enter the new AFE’s profile details

 

4.Click Submit.

How to Edit an AFE

Permission:Corporate Modify

Steps:

1.Select Corporate > AFEs.

2.Click Apply Filter.

 

3.Click the AFE Number to edit the AFE’s profile details.

4.Click Submit.

How to Delete an AFE

Permission:Corporate Modify

Steps:

1.Select Corporate > AFEs.

2.Click Apply Filter.

 

3.Click the Delete icon next to the AFE number you want to delete.

4.Click OK to confirm.

How to Export AFE Data

Permission:Corporate Modify

Steps:

1.Select Corporate > AFEs.

2.Click Apply Filter to display all AFEs.

 

3.Click Export Results As XLSX or Export Results As CSV.

4.Navigate to the folder on your computer where you want to save the file, and enter the filename.

5.Click Save.

AFE Profile Details

·AFE Number: must be unique within your list of AFEs.

·Property: associated property that will be displayed on any invoices referencing the AFE.

Note:If an invoice references both an AFE and a cost center, and both cost objects are associated with a property, the AFE’s property will be displayed on the invoice.

·Type: AFE type previously configured under Corporate > Reference Data.

·Active: setting that allows invoices to be coded to this AFE.

·Sites: sites that can make reference to this AFE.

Note:By default, all current and future sites can refer to any cost object. To restrict the cost object to specific sites, click the Edit icon.

·Field/Lease: options that are defined under Corporate > Reference Data.

Note:This option is only displayed on invoices if the coding is done by the supplier. If the company preference Restrict Field/Lease by AFE or Cost Center is enabled, only the field/lease associated with the specified AFE or cost center can be referenced on the invoice.

·PO Required: setting that determines whether detail or itemized image invoices referencing this AFE must also reference an existing OpenInvoice purchase order. Select Not Required, Required, or Required when CC is not specified.

·Active Date, Inactive Date, and Closeout Date: specifies when AFEs can be referenced (before the Active date, after the Inactive date, and after the Close out date).

Cost Center Profile Details

·Cost Center Number: must be unique within your list of cost centers.

·Property: location that appears on any invoices referencing this cost center.

Note:If an invoice references both an AFE and a cost center, OpenInvoice will use the AFE property on the invoice. If the AFE is not associated with a location, the Property field will be blank. The cost center’s location only appears when the cost center is not associated with an AFE.

·Type: cost center type previously configured under Corporate > Reference Data.

·Active: setting that allows invoices to be coded to this cost center.

·Sites: sites that can reference this cost center.

Note:By default, all current and future sites can refer to any cost object. To restrict the cost object to specific sites, click the configure site mapping icon.

·Field/Lease: options defined under Corporate > Reference Data.

Note:This option only appears on invoices if the coding is done by the supplier. If the company preference Restrict Field/Lease by AFE or Cost Center is enabled, only the field/lease associated with the specified AFE or cost center is referenced on the invoice.

·PO Required: setting that determines whether detail or itemized image invoices referencing this cost center must also reference an existing OpenInvoice purchase order. Select Not Required, Required, or Required when AFE is not specified.

Associating Locations to a Cost Object

This section includes procedures for adding, editing, or deleting a location for a cost object.

How to Add a Location

Permission:Corporate Modify

Steps:

1.Select Corporate > Cost Centers or Corporate > AFEs.

2.Perform a search for the cost object.

Note:If criteria is unknown, click Apply Filter to return a complete list.

3.Click a Cost Center or AFE number.

4.Click Add Location to enter location information, then click Submit.

Note:Select the Default checkbox if you are associating multiple locations to the cost object and would like this location to be the default. This default location appears on documents.

If an invoice references an AFE that is associated with a cost center, only the AFE location appears on the document. If no location is associated with the AFE, the Location field is blank regardless of the cost center’s associated location.

5.Click Submit or Cancel to return to the list page.

Note:All changes made to locations are saved even if you click Cancel.

How to Delete a Location

Permission:Corporate Modify

Steps:

1.Select Corporate > Cost Centers or Corporate > AFEs.

2.Perform a search for the cost object.

Note:If criteria is unknown, click Apply Filter to return a complete list.

3.Click a Cost Center or AFE number.

Cost Object Location Details

·Location: physical location associated with the cost object.

·Default: location that appears on documents when multiple locations are associated to the cost object. The default location is marked with a triangle.

Note:If there are multiple locations associated with a cost object without a default location configured, the Location field on the document will be blank unless the cost object and associated location is selected using search.

·Active: setting that allows locations to be automatically applied to invoices. If this checkbox is not selected, users can manually enter the location on the invoice coding page.

Associating a Cost Object with a Site

How to Modify the Sites Mapped to a Cost Object

Permission:Corporate Modify

Steps:

1.Select Corporate > Cost Centers or Corporate > AFEs.

2.Perform a search for the cost object.

Note:If criteria is unknown, click Apply Filter to return a complete list.

3.Do one of the following:

§In the row of the cost object you wish to modify, click the Configure site mapping icon in the Sites column.

§Click the cost object number, then click the Configure site mapping icon next to Sites.

 

4.Add or remove sites from the Mapped Sites box by selecting a site and clicking Add < or > Remove.

Note:To select multiple items at once, hold the Ctrl key and select the site names. Alternatively, hold the Shift key to select a range of site names.

 

5.Click Submit to save your changes.

Note:If you are returned to the cost object profile page, your mapped sites changes have been saved. Click Submit or Cancel on this page to return to the list page.
