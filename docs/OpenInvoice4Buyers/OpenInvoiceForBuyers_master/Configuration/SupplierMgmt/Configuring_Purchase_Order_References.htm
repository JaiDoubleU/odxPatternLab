Configuring Purchase Order References

By default, if valid purchase orders exist in OpenInvoice for the supplier, the referenced purchase order on the invoice must be one of the purchase orders in the platform. You have the option to allow suppliers to enter purchase orders that are not in the platform.

How to Configure Supplier Sites for External Purchase Order References Across Multiple Companies

Permission:Corporate Partnership

Steps:

  1. Select Corporate > Suppliers.

  2. Do one of the following:

    3. §Click Search Suppliers in the Tasks section and perform a search.

    §Click the Enabled counter in the Suppliers section.

    Note:Apply a filter to narrow the list of supplier sites. Sort the list by clicking a column header.

  3. Select the checkbox for each site.

    4. Note:To select all sites, select the checkbox in the header.

          

     

          

  4. Click the down arrow on the Select an Action button.

  5. Select either Allow External PO or Don’t Allow External PO.

  6. To configure this setting for supplier sites that are on another page in the list, navigate to the page and repeat the steps above.

How to Configure Supplier Sites for External Purchase Order References Within a Single Company

Permission:Corporate Partnership

Steps:

  1. Select Corporate > Suppliers.

  2. Do one of the following:

    3. §Click Search Suppliers in the Tasks section and perform a search.

    §Click the Enabled counter in the Suppliers section.

    Note:Apply a filter to narrow the list of supplier sites. Sort the list by clicking a column header.

  3. Click the supplier site.

    4.  

    Step 3: Configure external purchase order referencing for sites from a single supplier company.

  4. On the Supplier Configuration page, click Edit below the Settings and Early Pay Programs sections.

  5. In the Settings section, select the Allow External PO checkbox.

  6. Select the checkbox for each site to be modified.

    7. Note:Select the checkbox in the list header to select all sites in the list.

    All settings, including the supplier number, will be applied to the selected sites.

  7. Click Apply to Selected.

  8. Click Done.
