Managing Unregistered Suppliers in OpenInvoice

Unregistered supplier information can be entered into the platform via image invoice import, purchase order import, TSV file upload, or manual entry. The unregistered supplier’s name and supplier number are referenced from the image invoice import or purchase order import to create a site and company profile. Unregistered supplier profiles can be managed directly in OpenInvoice or via file upload.

Note:To delete individual sites of an unregistered supplier company, refer to Delete Unregistered Supplier Sites.

When an unregistered supplier has become a registered supplier, you have the ability to merge the supplier records and automatically delete the unregistered supplier record. The merge process transfers all existing invoices and purchase orders (including fulfillment history) from the unregistered supplier record to the registered supplier record. The registered supplier will have full visibility into the transferred invoices and purchase orders. When the supplier references a transferred purchase order on new invoices, fulfillment history from invoices you processed on their behalf will be considered.

How to Manage Unregistered Supplier Sites via File Upload

Permission:Corporate Partnership

Note:If you are deleting a supplier site that you have created in error, all invoices associated with the site must be corrected and moved to another supplier site.

Steps:

  1. Select Corporate > Suppliers.

  2. Click Export Unregistered Suppliers and save a copy of your unregistered suppliers list.

  3. In the file, enter the new supplier data, and save your file in TSV format. For more information about the column headings, refer to Unregistered Supplier File Template .

    4. Note:Multiple suppliers can be added to the template.

    To ensure unregistered suppliers can be enabled, enter all required data values. Enter 1 in the Site_Active column.

    If you use OpenInvoice to make payments and the VALIDATE_FOR_PAYMENT column has a value of 1, OpenInvoice will automatically attempt to validate the supplier for payment.

  4. In OpenInvoice, click Import Unregistered Suppliers.

  5. Do one of the following:

    6. §To upload a file saved on your desktop, select File as the import method, click Choose File, navigate to the file on your computer, and click Open.

    §To copy the contents of a spreadsheet, select Copy/Paste as the import method, copy the information in your file, and paste it into the box.

     

  6. Click Upload.

How to Merge Registered and Unregistered Supplier Records

Permission:Corporate Partnership

Prerequisite:The supplier records to be merged have a matching supplier number combination.

The unregistered supplier does not have invoices paid through OpenInvoice.

OpenInvoice Support has enabled the supplier merge feature.

Note:The transfer of documents cannot be reversed.

A supplier merge request can also be initiated from the Supplier Configuration page when a user is enabling a supplier site with a matching supplier number combination.

Steps:

  1. Select Corporate > Suppliers.

  2. In the Tasks section, click Merge Suppliers.

    3. Note:By default, all new or in-progress merge requests are listed. To modify the list of visible merge requests, select an option from the drop-down above the Action column.

    Sort the list by clicking a column header.

  3. Click Create Merge Request.

  4. Search for matching suppliers by Supplier Name or Supplier Number.

    5.  

  5. Click the Merge icon next to the supplier you want to merge.

  6. In Merge Supplier Wizard, verify the unregistered supplier site you are merging by clicking the Purchase Orders and Invoices counters to view all documents associated with the supplier.

    7. Note:A warning will appear if there are duplicate purchase order numbers found between the suppliers. The unregistered supplier’s purchase order will be cancelled before merging. A comment will be added to the purchase order stating why it was cancelled. Invoices referencing the cancelled purchase order can continue through workflow.

  7. Click Next.

  8. Select the registered Supplier Site, then click Next.

  9. Repeat steps 6 to 8 for each unregistered supplier site.

  10. Verify where the unregistered supplier site(s) will be merged, then click Done.

  11. Click Yes, I agree to confirm.

Note:Merge requests are processed daily. To cancel a merge request before it is processed, click the Action icon in the Pending or In-Progress list of merge requests and select Cancel.

Once the merge is complete, the unregistered supplier record will automatically be deleted.
