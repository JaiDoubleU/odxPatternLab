Sending a Message to Suppliers

A message can be sent to one or more of your enabled registered suppliers. The message is sent to all subscribed users in the supplier company. If the supplier has multiple sites, the message sent to the supplier will include the specific sites that you selected.

How to Send a Message to Suppliers

Permission:Corporate Partnership

Prerequisite:Recipient supplier sites must be registered with transactions enabled.

Steps:

  1. Select Corporate > Suppliers.

  2. Do one of the following:

    3. ยงClick Search Suppliers in the Tasks section and perform a search for enabled suppliers.

    ยงClick the Enabled counter in the Suppliers section.

  3. If necessary, apply a filter to narrow the list.

  4. Select the checkbox next to the supplier sites that should receive your message.

    5. Note:To select all supplier sites, select the checkbox in the header.

    Only the registered supplier sites visible on the current page will be messaged.

  5. Click the down arrow on the Select an Action button and select Message.

  6. Enter the subject and message.

    7. Note:The last message sent during the current session prepopulates the message fields. Edit the message as necessary.

          

     

          

  7. Click Send.

  8. To send the same message to supplier sites that are on another page in the list, navigate to the page and repeat steps 4 - 7.
