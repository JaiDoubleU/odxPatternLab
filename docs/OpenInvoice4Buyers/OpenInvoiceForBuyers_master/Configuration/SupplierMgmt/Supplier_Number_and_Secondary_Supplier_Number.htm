Supplier Number and Secondary Supplier Number

As the buyer company, you assign supplier numbers and secondary supplier numbers in your back office system based on how you transact with supplier companies. These numbers are used to identify both the supplier company and the specific supplier sites in all document transfers between your company and OpenInvoice (purchase orders you submit to OpenInvoice, invoice exports from OpenInvoice to you, etc.).

To enable a supplier, a supplier number is required and a secondary supplier number is optional. The numbers can be assigned to multiple sites and may span multiple companies; however, the combination of supplier number and secondary supplier number must uniquely identify the supplier company.

Note:A registered and unregistered supplier company can share the same combination of supplier number and secondary supplier number. If this happens, the registered supplier will take precedence for all document imports.

OpenInvoice only uses the supplier number when performing a duplicate invoice check. If a supplier number is shared between two supplier companies and the companies are distinguished using the secondary supplier number, the two companies cannot submit the same invoice number.

The purchase order import file only allows one number to be specified—either the supplier number or the secondary supplier number. During the import process, the system will attempt to match the number provided in the purchase order with a supplier number that you have entered in OpenInvoice. If multiple suppliers or no supplier companies are a match, the number provided in the purchase order is then compared to the secondary supplier numbers you have entered in OpenInvoice.

·If the specified number matches a single supplier company, the purchase order is associated to all sites with the matching number.

·If the specified number matches multiple supplier companies, the purchase order import will fail.

·If the specified number does not match any supplier companies in OpenInvoice, a new unregistered supplier company will be created in OpenInvoice.

Note:If you are using supplier numbers to configure Invoice Workflow Rules refer to Invoice Workflow Rules.
