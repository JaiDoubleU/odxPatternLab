Inviting the Supplier to Register

How to Attach a Personalized Message to the Registration Invitation

Permission:Corporate Partnership

Steps:

  1. Create a personalized message and save it in PDF format.

  2. Select Corporate > Suppliers.

  3. Click On-Ramp Suppliers.

  4. Click On-Ramping Settings.

  5. Click Choose File, navigate to the file on your computer, and click Open.

    6. Note:This file will replace any previously uploaded file.

     

  6. Click Upload.

Note:The same message will be attached to all future supplier invitations unless you delete or replace it.

How to Send a Single Registration Invitation

Permission:Corporate Partnership

Steps:

  1. Select Corporate > Suppliers.

  2. Click On-Ramp Suppliers.

  3. Click the Supplier Name.

    4.  

  4. Click Send Invite.

Note:If the Send Invite button is not active, you are not the owner of this request or the supplier contact has unsubscribed from OpenInvoice registration emails.

Note:The email invitation to suppliers includes a self-registration wizard allowing them to add sites, taxes, users, and a catalog of products and services.

How to Send Multiple Registration Invitations

Permission:Corporate Partnership

Steps:

  1. Select Corporate > Suppliers.

  2. Click On-Ramp Suppliers.

  3. Expand the Batch Update section.

  4. Select the Send Invitation checkbox.

    5.  

  5. Select the checkboxes next to the suppliers you want to invite.

  6. Click Apply to Selected.

Note:Unsubscribed supplier contacts will not receive an email. After the batch of emails, a message will display the number of emails that were successfully sent.

Note:The email invitation to suppliers includes a self-registration wizard allowing them to add sites, taxes, users, and a catalog of products and services.
