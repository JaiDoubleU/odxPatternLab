Approval Types

If using the GL Based Approvals method, configuring multiple approval types allows you to restrict invoice approval by the general ledger coding assigned to the invoice. The Total approval type is used to limit the total invoice amount that a user can financially approve. Once defined in OpenInvoice, approval types are mapped to the chart of accounts using one of the following methods:

·Map major account codes to approval types by assigning a major and all associated minors to the approval type.

·Assign an approval type to a major/minor combination. Each major/minor combination can be assigned to only one approval type.

How to Configure Approval Types for a Single Record

Permission:Corporate Modify

Steps:

  1. Select Corporate > Reference Data.

  2. Click Approval Type.

  3. Do one of the following:

    4. §Click Add Record to add a new approval type.

    §Click the Approval Type or Description to edit the record.

    §Click the Delete icon to delete the record.

     

  4. Click Submit.

How to Configure Approval Types for Multiple Records

Permission:Corporate Modify

Steps:

  1. Select Corporate > Reference Data.

  2. Click Approval Type.

  3. Click Export to download and save a copy of your existing records.

  4. In the file, modify your existing records and/or add new custom reference data. For more information, refer to Approval Types Template .

    5. Note:Enter Add, Edit, Delete, or Leave in the Action column. These values are case-sensitive.

    Do not modify or delete values in the Row#(DO NOT MODIFY/DELETE) column.

  5. In OpenInvoice, click Import.

  6. Do one of the following:

    7. §To upload a file saved on your desktop, select File as the import method, click Choose File, navigate to the file on your computer, and click Open.

    §To copy the contents of a spreadsheet, select Copy/Paste as the import method, copy the information in your file, and paste it into the box.

  7. Click Upload.

  8. Click Submit.

How to Map Major Accounts to Approval Types

Permission:Corporate Modify

Prerequisite:Approval types must be created.

The chart of accounts must be mapped.

Steps:

  1. Select Corporate > COA Setup.

  2. Click Account Approvals.

  3. Click the approval type.

  4. Add or remove accounts to be mapped by selecting the account name and clicking Add > or < Remove.

    5. Note:To select multiple items at once, hold the Ctrl key and select the items. Alternatively, hold the Shift key to select a range of items.

     

  5. Click Submit.

How to Assign an Approval Type to a Major/Minor Pair

Permission:Corporate Modify

Prerequisite:Approval types must be created.

Steps:The chart of accounts must be mapped.

  1. Select Corporate > COA Setup.

  2. Click Account Approvals.

  3. Click Map By Account.

  4. Enter the major code or perform a search.

  5. Enter the minor code or perform a search.

  6. Select an approval type.

    7. Note:Select None to remove the approval type assignment from the major/minor.

    If an account is already mapped to another approval type, it will be removed from that approval type and added to the new one.

  7. Click Submit.

Additional Information

·If an entire major code is added to an approval type, and then one or more major.minor combinations have been moved from that approval type to another, the Mapped Accounts column will list the remaining major.minor codes individually. However, in the Major Account Mapping for Approval Type page, the original approval type will still look like the major is mapped to it.

·Not all major.minor combinations can be moved from a mapped major to another approval type. At least one must remain in the original approval type. If a user tries to remove the final major.minor combination, they will see the error message Approval type is already mapped to the account.

·Any major.minor combinations that were moved out of the original approval type cannot be moved back into it. The confirmation prompt will say Account is already mapped to an approval type. Continue? If OK is selected, the error message Approval type is already mapped to the account will appear, even though it has been moved away. If just the major.minor combination was mapped to the approval type originally, it can be moved in and out of the approval type without a problem. If the major.minor combination needs to be moved back into the approval type but there is an error message, click the approval type name, remove the major mapping and then re-add the necessary major.minor combinations.

·To remove individual major.minor combinations, the entire major account must be removed. Any required major.minor combinations can then be added back individually using the Map By Account method.
