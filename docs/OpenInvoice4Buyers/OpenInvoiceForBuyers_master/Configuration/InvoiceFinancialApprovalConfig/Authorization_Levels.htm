Authorization Levels

Authorization levels establish a financial limit and currency for each approval type. If using the Total Invoice Amount approval method, only authorization levels associated with the Total approval type is used.

A user is able to financially approve an invoice if one of the userâ€™s assigned authorization levels has an approval type associated with the invoice. To perform a final financial approval, the user must have sufficient financial limit, in the currency of the invoice, for all approval types associated with the invoice.

Refer to Granting Financial Authority for instructions on how to assign authorization levels to users.

How to Add a New Authorization Level

Permission:Membership Manage Auth

Prerequisite:If using approval types, they must be created and mapped to the chart of accounts by a user with the Corporate Modify permission.

Steps:

  1. Select Membership > Authorization Levels.

  2. Click Add Authorization Level.

    3.  

  3. Enter a level name.

    4. Note:Level names can be repeated but must be unique for each approval type and currency pair.

  4. Select an approval type and currency.

  5. Enter an authorization limit or leave the field blank to indicate an unlimited authorization level.

    6. Note:Only one authorization level per approval type and currency pair can be unlimited.

  6. Click Submit.

 

How to Modify an Existing Authorization Level

Permission:Membership Manage Auth

Prerequisite:If using approval types, they must be created and mapped to the chart of accounts by a user with the Corporate Modify permission.

Steps:

  1. Select Membership > Authorization Levels.

  2. Click the level name.

    3.  

  3. Edit the authorization level profile.

  4. Click Submit.

How to Delete an Existing Authorization Level

Permission:Membership Manage Auth

Prerequisite:If using approval types, they must be created and mapped to the chart of accounts by a user with the Corporate Modify permission.

Steps:

  1. Select Membership > Authorization Levels.

  2. Click the Configure Users icon and remove all users from the authorization level.

  3. Click the Delete icon.

Note:The Delete icon only appears if there are no users in the authorization level.
