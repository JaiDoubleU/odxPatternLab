Number of Financial Approvals Required

When a user financially approves an invoice, OpenInvoice determines if additional approvals are required or if the user is able to perform the final approval. To determine if you are able to perform a final financial approval, the answer must be Yes to all of the following questions; if the answer is No for any question, the invoice must be forwarded to another user for additional financial approval.

·Has the invoice received the minimum number of signatures needed for final financial approval?

·If you are the requisitioner, are requisitioners allowed to perform final financial approvals?

·Do you have a sufficient financial limit for all approval types associated with the invoice?

·Was the invoice submitted by another user? (You cannot final approve an invoice you submitted.)

If the user is able to perform the final financial approval in the Code then Approve or Approve Only invoice workflow processes, the invoice status is changed to Approved and the next invoice on the invoice list page is automatically displayed. If the user is able to perform the final financial approval in the Approve then Code workflow, the invoice status does not change and the user must forward the invoice to another user for code verification. Once the invoice is forwarded for code verification, the next invoice on the invoice list page is automatically displayed.

A minimum number of signatures can be specified for final financial approval. The invoice must be forwarded to another user for approval if the minimum number of signatures has not been met.

By default, requisitioners are allowed to perform a final financial approval. Enable the company preference Invoice Approval - Requisitioner May Not Final Approve Invoice if you require the final approval to be performed by a different user.

How to Configure the Minimum Number of Signatures Required

Permission:Corporate Modify

Steps:

  1. Select Corporate > Workflow Configuration.

  2. Click Configure Number of Approvals Rules.

  3. Click Export to download and save a copy of the template.

  4. In the file, modify the rules regarding the minimum number of signatures required. For more information about the column headings, refer to Number of Financial Approval Template.

    5. Note:The modified file will completely replace all existing rules. Any records not in the file will be deleted during the import.

    The rules are evaluated in order from the top to bottom of the file. The first rule that the invoice qualifies for will be used.

    Values in the Amount To column are not required. If the Amount To is blank, there is no upper limit to the rule.

  5. In OpenInvoice, click Import.

  6. Do one of the following:

    7. §To upload a file saved on your desktop, select File as the import method, click Choose File, navigate to the file on your computer, and click Open.

    §To copy the contents of a spreadsheet, select Copy/Paste as the import method, copy the information in your file, and paste it into the box.

  7. Click Upload.
