Invoice Approval Method and Currency Awareness

The Invoice Approval Method determines the type of authorization level required by a financial approver when approving an invoice. The approval method is configured by an Oildex-controlled preference with two options:

·GL Based Approvals: to perform the final financial approval, the user must have an authorization level of sufficient dollar value for every general ledger code on the invoice. The user must have an authorization level for at least one general ledger code in order to add their signature (approval) to the invoice and forward for additional approval. By default, OpenInvoice uses this option.

·Total Invoice Amount: all general ledger codes on the invoice are ignored for approval purposes and, to perform the final financial approval, the user must have a Total authorization level of sufficient dollar value to cover the invoice amount.

Currency awareness determines whether or not the invoice currency is considered for financial approval purposes. The currency awareness is configured by an Oildex-controlled preference with two options:

·Currency Aware: to add their signature (approval) to an invoice, a user must have an authorization level in the currency of the invoice.

·Unit Based: all authorization levels are defined in the “preferred” currency and the currency of the invoice is ignored. A user is able to add their signature (approval) to an invoice if they have an authorization level in the preferred currency.

By default, final financial approval requires a user to have an authorization level of sufficient dollar value to cover the cost of the products and services as well as the taxes. OpenInvoice can be configured to exclude a tax type from the approval process. If the tax type is excluded from the approval process, the user only needs an authorization level of sufficient dollar value to cover the cost of the products and services.

How to Configure Tax Type Approval Requirements

Permission:Corporate Modify

Steps:

  1. Select Corporate > COA Setup > Tax Type Profile

  2. Select the Exclude From Approval Requirements checkboxes as needed for each tax type.

  3. Click Submit.
