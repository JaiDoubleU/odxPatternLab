Using Color

Supporting Principles
  • Less is More - use color sparingly and consider color in the context of the entire application, not just on a single page
  • W3C Compliance - Use bootstrap CSS classes to ensure compliance with color contrast ratio standards for users with visual impairments - see the W3C standards for color contrast.
  • Reliance on Color - don't rely solely on difference in color to convey information
